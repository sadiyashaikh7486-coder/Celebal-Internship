# Shopping Dataset Analysis

## Author
Sadiya Shaikh

## Objective
Perform exploratory data analysis (EDA), data cleaning, feature engineering, and visualization on a shopping dataset.

## Dataset
combined_dataset.csv

## Tasks Performed

### Data Exploration
- Loaded dataset using Pandas
- Checked shape, columns, and data types
- Generated summary statistics

### Data Cleaning
- Identified missing values
- Filled missing values appropriately
- Checked and removed duplicate records

### Feature Engineering
- Created price_difference column
- Created popularity_score column

### Analysis
- Univariate Analysis
- Bivariate Analysis
- Category-level Analysis

### Visualizations
- Price Distribution
- Rating Distribution
- Price vs Rating
- Category Analysis

## Key Insights
- Dataset contains 1000 products.
- Missing values were handled successfully.
- No duplicate records were found.
- Products generally have high ratings.
- Some categories contain significantly more products than others.

- Python
- Pandas

- Matplotlib
- Seaborn
- Jupyter Notebook
